2D Frame Transform Playground (Translation + Rotation)

Drag User origin (square) to translate. Drag Rotation handle (circle on +X axis) to rotate. Drag the Point (filled circle) to change pu. Values update live. 

    

      World ↔ User formulas:
      
p_w = R(θ) p_u + t
      
p_u = R(θ)^T (p_w - t)
